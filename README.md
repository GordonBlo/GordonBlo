## AI-Assisted Development Workflow

I use AI tools as development assistants, but I do not treat generated code as automatically correct.

My workflow is:

1. Define the task and scope clearly
2. Let the AI suggest or generate code
3. Review the structure and logic
4. Test the API or feature manually
5. Debug errors step by step
6. Commit only working, understandable code
7. Document what was built and why

This helps me learn faster while still building real understanding of architecture, backend logic, and code quality.
